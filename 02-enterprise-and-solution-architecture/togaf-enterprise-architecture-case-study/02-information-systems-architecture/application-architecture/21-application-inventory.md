# Application Inventory

## Purpose
Create a complete view of all applications supporting the enterprise.

## Application Categories
- ERP Systems  
- CRM Systems  
- Manufacturing and MES  
- Supply Chain and Logistics  
- Finance and Accounting  
- HR Systems  
- Analytics and BI  
- Custom and Legacy Apps  

## Inventory Attributes
For each application:
- Business owner  
- Technology owner  
- Supported capabilities  
- Users and regions  
- Cost and criticality  
- Integration points  

## Observations
- Multiple ERPs across regions  
- Overlapping CRM tools  
- High number of low-value legacy apps
